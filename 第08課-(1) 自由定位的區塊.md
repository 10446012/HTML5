## (1) 自由定位的區塊

#####檔案放置方式:
```
   | 
   |___<css>
   |     |___ main.css
   |
   |___<images>
   |     |___ photo.jpg  (300像素*450像素) 
   |   
   |___index.html   
```


#####檔案名稱: index.html 
```html
<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>測試網頁</title>
<!-- 加入 main.css -->
<link href="css/main.css" rel="stylesheet" type="text/css" />
<link href="https://fonts.googleapis.com/css?family=Reenie+Beanie" rel="stylesheet">
</head>
<body>    
    <div class="container">

        <div class="photo">
        </div>

        <div class="text">           
            <span class="large">厚</span>重的窗簾阻隔了室外和煦的陽光, <br/>
            還有徐徐的涼風.<p/>
            那和記憶中某日在花蓮感受的陽光一樣, 明亮又溫暖, <br/>

    </div>

```


#####檔案名稱: css/main.css
```css
@charset "utf-8";

/*---------------------------*/
/* 引用仿宋體                  */
/*---------------------------*/
@import url(http://fonts.googleapis.com/earlyaccess/cwtexfangsong.css);


/*---------------------------*/
/* 定義網頁                   */
/*---------------------------*/
html, body{
    font-size:17px;
    /*設定使用字體*/
    font-family: 'Reenie Beanie', cursive, 'cwTeXFangSong';
    font-weight: 280;   
    background:#000;    
}



/*---------------------------*/
/* 定義容器                   */
/*---------------------------*/
div.container{
    position:relative;
    width:800px;      
    margin:0px auto 0px auto;   
    background:#000;	
}


/*---------------------------*/
/* 定義照片                   */
/*---------------------------*/
div.photo{
    position:absolute;
    background:url(../images/photo.jpg);
    width:300px;
    height:450px;
    left:50px;
    top:150px;;
}


/*---------------------------*/
/* 定義文字                   */
/*---------------------------*/
div.text{
    position:absolute;
    left:330px;
    top:230px;
    color:#fff;
    font-size:18px;
    line-height:28px;	
    letter-spacing:1px;
}

div.text span.large{
    font-size:24px;
}

div.text span.small{
    font-size:15px;
}
```
